# Financial Performance & Target Tracking Dashboard
A professional financial analytics project designed to monitor revenue growth, customer portfolio health, and target achievement across multiple international markets. This project provides a high-level view of business performance, comparing current year-to-date (YTD) results against historical data and set goals.

## Project Overview
This dashboard analyzes a portfolio of 105 unique customers and over €2.11M in total revenue. It is engineered to provide multi-dimensional visibility into revenue streams, allowing stakeholders to evaluate performance by country, industry, and company size.

## Key Performance Indicators (KPIs)
Total Revenue (€2.11M): Consolidated revenue tracking across all active accounts and regions.

Target Achievement (108%): A dynamic comparison of actual revenue vs. projected targets (exceeding the €1.95M goal).

Revenue per Customer (€20K): Average value generated per account to assess portfolio quality.

Year-Over-Year (YoY) Growth: Comparative analysis of monthly performance between 2022 and 2023.

## Technical Features
Revenue Accumulation Logic: Developed a monthly accumulation model to track YTD performance against target burn rates.

Market Segmentation: Built deep-dive views into 5+ countries (Germany, USA, France, Italy, Norway) and 5+ industries (Chemical, Food, Beverage, etc.).

Company Size Analysis: Categorized performance by client scale (from 1-10 to 100-1000 employees) to identify the most profitable customer segments.

Variance Tracking: Integrated automated logic to highlight months where current revenue fell below target or previous year benchmarks.

## Tools Used
Data Analysis: Excel (Advanced Pivot Tables, Power Query), Data Modeling.

Financial Modeling: Target vs. Actual Variance Analysis, YTD Accumulation.

Reporting: Interactive Dashboard Design with slicers for Year, Country, and Industry.
